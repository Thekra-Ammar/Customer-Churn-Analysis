# 📊 Customer Churn Analysis (Power BI + Python + Machine Learning)

## 📌 Project Overview
This project analyzes customer churn in the telecommunications sector using Power BI, Python, and Machine Learning.

The objective is to understand churn behavior and predict customers at risk of leaving.

---

## 📂 Dataset
- 6,600+ customers
- 29 features
- Includes demographic, contract, usage, and churn-related information

---

## 🧹 Data Preparation
- Data cleaning
- Creation of new variables (Demography, Contract Category, Consumption Level)
- Data transformation for analysis and modeling

---

## 📊 Exploratory Data Analysis (EDA)
Key insights:
- Churn rate: **26.86%**
- Month-to-Month contracts have highest churn
- Certain segments and regions (e.g., California) show higher risk
- Main reasons: price, service quality, competition

## 🤖 Machine Learning Model
- Model: Random Forest
- Pipeline: preprocessing (numerical + categorical)
- Evaluation:
  - Accuracy: **88.7%**
  - ROC AUC: **0.935**
  - Recall (churn): **73%**

---

## 📌 Business Insights
- Short-term contracts increase churn risk
- Targeted retention strategies can reduce customer loss
- Predictive model helps identify at-risk customers

---

## 🛠 Tools Used
- Power BI
- Python (Pandas, Scikit-learn)
- Machine Learning (Random Forest)

---

## 📌 Conclusion
This project combines data analysis and predictive modeling to provide actionable insights for improving customer retention.
